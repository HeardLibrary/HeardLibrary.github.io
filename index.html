"Getting Started With..." Workshops

The library provides hands on workshops to introduce students, faculty and staff to emerging technologies that may be useful for research.

Learn more

Digital Scholarship Working Groups

The library offers four working groups that study and implement emerging methods in digital scholarship, including data curation, geographic information systems, linked data, the semantic web, and XQuery.

Learn more

Cultural Heritage at Scale

The library sponsors an annual symposium titled *Cultural Heritage at Scale* to foster conversation about the appropriate tools and methods to conduct cultural heritage preservation and access at web-scale.

Learn more

Mapping Berlin

The library maintains the *Mapping Berlin* project, which uses open source technology to produce a "thick mapping" of the history of Berlin in the 20th century.

Learn more

Network Analysis using Neo4j

The library hosts Neo4j graphgists to analyze and display interconnections in many kinds of datasets.
